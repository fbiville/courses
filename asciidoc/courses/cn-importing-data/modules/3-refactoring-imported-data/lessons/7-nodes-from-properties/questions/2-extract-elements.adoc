[.question]
= 2. Processing a list

What Cypher keyword is used to expand the elements of a list as rows during a query?

* [ ] `FOR EACH`
* [x] `UNWIND`
* [ ] `EXTRACT`
* [ ] `ITERATE`

[TIP,role=hint]
.Hint
====
With this keyword, you are unwinding the elements of the list.
====

[TIP,role=solution]
.Solution
====
The correct answer is:  `UNWIND`
====
