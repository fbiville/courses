[.question]
= Writing to Neo4j

Which method would you use to write data to Neo4j?

- [ ] `ExecuteNeo4jWrite()`
- [ ] `neo4j.ExecuteRead()`
- [*] `neo4j.ExecuteWrite()`
- [ ] `neo4j.Write()`


[TIP,role=hint]
.Hint
====
You are looking to _execute_ a _read_ query against the _neo4j_ database.
====

[TIP,role=solution]
.Solution
====
The answer is `neo4j.ExecuteWrite()`
====
