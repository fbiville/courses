[.question]
= Separating lists

What Cypher clause do you use to separate list elements?

* [ ] `MATCH`
* [ ] `WITH`
* [ ] `RETURN`
* [x] `UNWIND`

[TIP,role=hint]
.Hint
====
This clause operates on a list to return a row value.
====

[TIP,role=solution]
.Solution
====
The clause used to separate a list into individual elements is called `UNWIND`.
====
