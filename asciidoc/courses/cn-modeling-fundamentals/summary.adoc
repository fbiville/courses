= Course Summary

In this course, you have learned how to:

* Describe what is a graph data model?
* Model nodes and createnodes for an instance model.
* Model relationships and create relationships for an instance model.
* Test the graph data model.
* Refactor a graph data model to add labels.
* Eliminate duplicate data in the graph.
* Add specific relationship types.
* Add intermediate nodes.
